# Posts Board React Demo

A simple React application for managing and displaying posts.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- View posts
- Create new posts
- Delete posts
- Simple and clean UI

## Tech Stack

- React
- Vite
- CSS
